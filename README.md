# BookApp Pro

A complete Android book browsing application built with Kotlin.

## Features

- User Authentication (Login/Signup)
- Google Books API Integration
- Real-time Book Search
- Category Filters
- Browsing History
- Beautiful Material Design 3 UI
- Collapsing Toolbar Detail Pages

## Setup Instructions

1. Open Android Studio
2. Select "Open Project"
3. Navigate to this folder
4. Wait for Gradle sync to complete
5. Run on emulator or device

## Technologies Used

- Kotlin
- Retrofit (Networking)
- Coroutines (Async)
- Glide (Image Loading)
- Material Design 3
- View Binding
- SharedPreferences

## API

Uses the free Google Books API - no API key required!

## Build

- Min SDK: 24
- Target SDK: 34
- Compile SDK: 34

Enjoy building with BookApp Pro!
